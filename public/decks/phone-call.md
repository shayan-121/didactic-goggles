# Phone Call Vocabulary

## General Nouns

| Dutch | English |
|-------|---------|
| de afspraak | the appointment |
| de receptie | the reception |
| de assistent(e) | the assistant |
| de dokter / de arts | the doctor |
| de huisarts | the GP / family doctor |
| de wachttijd | the waiting time |
| de bevestiging | the confirmation |
| het BSN-nummer | the citizen service number (BSN) |
| de geboortedatum | the date of birth |
| het telefoonnummer | the phone number |
| het e-mailadres | the email address |
| de locatie | the location |
| het hoofdmenu | the main menu |
| de spoedlijn | the emergency line |
| het recept | the prescription |
| de sms | the text message |

## IVR — Automated Menu

| Dutch | English |
|-------|---------|
| Welkom bij... | Welcome to... |
| Dit gesprek wordt opgenomen. | This call may be recorded. |
| Voor afspraken, toets 1. | For appointments, press 1. |
| Voor recepten, toets 2. | For prescriptions, press 2. |
| Voor de spoedlijn, toets 3. | For the emergency line, press 3. |
| Om het menu te herhalen, toets 9. | To repeat this menu, press 9. |
| Om terug te gaan naar het hoofdmenu, toets ster. | To go back to the main menu, press star. |
| Een moment geduld alstublieft. | Please hold. |
| We verbinden u door. | We will connect you. |
| De huidige wachttijd is ongeveer vijf minuten. | The current waiting time is approximately five minutes. |

## Conversation Phrases

| Dutch | English |
|-------|---------|
| Goedemorgen, u spreekt met Lisa. | Good morning, you are speaking with Lisa. |
| Hoe kan ik u helpen? | How can I help you? |
| Ik wil graag een afspraak verzetten. | I would like to reschedule an appointment. |
| Mag ik uw naam en geboortedatum? | Can I have your name and date of birth? |
| Mijn naam is... | My name is... |
| Mijn geboortedatum is... | My date of birth is... |
| Mag ik uw BSN-nummer? | Can I have your BSN number? |
| Ter verificatie van uw identiteit. | To verify your identity. |
| Ik zie uw afspraak staan. | I can see your appointment. |
| U heeft momenteel een afspraak op... | You currently have an appointment on... |
| Wilt u deze afspraak verzetten? | Would you like to reschedule this appointment? |
| Die dag komt mij niet meer uit. | That day no longer works for me. |
| Is er iets beschikbaar volgende week? | Is there anything available next week? |
| Laat me even kijken. | Let me check. |
| Welke heeft uw voorkeur? | Which would you prefer? |
| Ik zet dat voor u om. | I will reschedule that for you. |
| Vrijdag de dertigste om veertien uur. | Friday the thirtieth at fourteen hundred hours. |
| Klopt dat? | Is that correct? |
| Ja, dat klopt. | Yes, that is correct. |
| U ontvangt een bevestiging per e-mail en sms. | You will receive a confirmation by email and text message. |
| Is uw telefoonnummer nog hetzelfde? | Is your phone number still the same? |
| Alles is nog hetzelfde. | Everything is still the same. |
| Is er nog iets anders waarmee ik u kan helpen? | Is there anything else I can help you with? |
| Nee, dat was alles. Hartelijk dank. | No, that is everything. Thank you very much. |
| Graag gedaan. Fijne dag. Tot ziens. | You are welcome. Have a good day. Goodbye. |

## Key Verbs

| Dutch | English |
|-------|---------|
| verzetten | to reschedule |
| annuleren | to cancel |
| bevestigen | to confirm |
| doorverbinden | to connect / transfer a call |
| wachten | to wait / to hold |
| intoetsen | to press / to key in |

## Useful Questions

| Dutch | English |
|-------|---------|
| Ik wil graag een afspraak maken. | I would like to make an appointment. |
| Ik wil graag een afspraak annuleren. | I would like to cancel an appointment. |
| Ik bel om een afspraak te bevestigen. | I am calling to confirm an appointment. |
| Met wie spreek ik? | Who am I speaking with? |
| Kunt u dat herhalen? | Could you repeat that? |
| Kunt u langzamer spreken? | Could you speak more slowly? |
| Kunt u dat spellen? | Could you spell that? |
| Wat moet ik meenemen naar de afspraak? | What should I bring to the appointment? |
| Wat is het adres? | What is the address? |
| Is er parkeren beschikbaar? | Is there parking available? |
| Hoe laat moet ik er zijn? | What time should I be there? |
| Heeft u een tolk beschikbaar? | Do you have an interpreter available? |
| Ik spreek niet zo goed Nederlands. | I do not speak Dutch very well. |
| Spreekt u Engels? | Do you speak English? |
| Hoe lang is de wachttijd? | How long is the waiting time? |
