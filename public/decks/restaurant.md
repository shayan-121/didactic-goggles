|-------|---------|

## Conversation Phrases

| Dutch | English |
|-------|---------|
| Goedenavond! | Good evening! |
| Welkom! | Welcome! |
| Heeft u een reservering? | Do you have a reservation? |
| Nee, we hebben geen reservering. | No, we don't have a reservation. |
| Is er een tafel vrij voor twee personen? | Is there a table available for two? |
| Volgt u mij alstublieft. | Follow me please. |
| Is deze tafel goed voor u? | Is this table okay for you? |
| Ja, dit is prima, dank u. | Yes, this is fine, thank you. |
| Hier is de menukaart. | Here is the menu. |
| Kan ik u iets te drinken brengen? | Can I get you something to drink? |
| We nemen een fles stil water. | We'll have a bottle of still water. |
| Bent u klaar om te bestellen? | Are you ready to order? |
| Wat is de soep van de dag? | What is the soup of the day? |
| Ik begin met de soep. | I'll start with the soup. |
| Wat raadt u aan voor het hoofdgerecht? | What do you recommend for the main course? |
| De gegrilde zalm is erg populair. | The grilled salmon is very popular. |
| Het wordt geserveerd met... | It comes with... |
| Bevat het zuivel? | Does it contain dairy? |
| Zonder zuivel alstublieft. | Without dairy please. |
| Mijn vriend neemt de pasta. | My friend will have the pasta. |
| Wilt u de dessertkaart zien? | Would you like to see the dessert menu? |
| Hoe was alles? | How was everything? |
| Het was heerlijk, dank u. | It was delicious, thank you. |
| Wilt u koffie of thee bij uw nagerecht? | Would you like coffee or tea with your dessert? |
| Mag ik de rekening? | Can we have the bill? |
| Betaalt u samen of apart? | Would you like to pay together or separately? |
| Samen alstublieft. | Together please. |
| Apart alstublieft. | Separately please. |
| Kan ik met pin betalen? | Can I pay by card? |
| Het totaal is... | The total is... |
| Prettige avond! | Have a lovely evening! |
| Het was uitstekend. | It was excellent. |
| Goedenacht! | Goodnight! |
