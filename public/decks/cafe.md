|-------|---------|

## Conversation Phrases

| Dutch | English |
|-------|---------|
| Goedemorgen! | Good morning! |
| Welkom! | Welcome! |
| Wat kan ik voor u doen? | What can I do for you? |
| Wat mag het zijn? | What will it be? |
| Ik wil graag een cappuccino. | I'd like a cappuccino. |
| Ik neem een vers sinaasappelsap. | I'll have a fresh orange juice. |
| Ik neem het kaasbroodje. | I'll take the cheese sandwich. |
| Wilt u nog iets anders? | Would you like anything else? |
| Nee dank u, dat is alles. | No thank you, that's everything. |
| Wilt u binnen of buiten zitten? | Would you like to sit inside or outside? |
| Is de tafel bij het raam vrij? | Is the table by the window free? |
| Ja, ga uw gang. | Yes, go ahead. |
| Ik breng het naar uw tafel. | I'll bring it to your table. |
| Excuseer mij! | Excuse me! |
| Kan ik met pin betalen? | Can I pay by card? |
| Tik of steek uw kaart in. | Just tap or insert your card. |
| Prettige dag! | Have a nice day! |
| U ook! | You too! |

## Useful Questions

| Dutch | English |
|-------|---------|
| Heeft u een menukaart? | Do you have a menu? |
| Mag ik de menukaart zien? | Can I see the menu? |
| Mag ik de rekening? | Can I have the bill? |
| Mag ik ook een glas water? | Can I also have a glass of water? |
| Mag ik nog een koffie? | Can I have another coffee? |
| Welke sappen heeft u? | What juices do you have? |
| Is het vers geperst? | Is it freshly squeezed? |
| Welke broodjes heeft u? | What sandwiches do you have? |
| Wat raadt u aan? | What do you recommend? |
| Is er een dagschotel? | Is there a daily special? |
| Voor hier of meenemen? | For here or to go? |
| Voor hier alstublieft. | For here please. |
| Meenemen alstublieft. | To go please. |
| Heeft u iets zonder gluten? | Do you have anything gluten-free? |
| Ik ben allergisch voor... | I am allergic to... |
| Heeft u wifi? | Do you have wifi? |
| Wat is het wifi-wachtwoord? | What is the wifi password? |
| Hoe laat sluit u? | What time do you close? |
